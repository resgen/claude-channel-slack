# Projects

> **Template:** fill in each project you've routed through the orchestrator. Keep this in sync with `~/.claude/channels/slack/routes.json`.

## Example: Project Name

- **Slack channel**: `#channel-name`
- **Channel ID**: `C...`
- **Repo**: `/absolute/path/to/repo`
- **Repo GitHub**: (optional) `https://github.com/org/repo`
- **Purpose**: short description of what this project is and what kind of questions come through its channel
- **Key users**: who interacts with the bot in this channel
- **CLAUDE.md highlights**: the main workflows defined in the repo's CLAUDE.md (for your quick reference)
- **Notes**: anything else useful — recurring patterns, gotchas, links to other docs

---

## Add one of these per routed project

Duplicate the example block above for each entry in your `routes.json`. The orchestrator reads this file to answer "what are you watching?" queries in DM, and as a reference when onboarding new users to the system.
